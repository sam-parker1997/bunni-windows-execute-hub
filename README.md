## Overview

Bunni is a Windows-focused Roblox script executor that runs Lua scripts through a desktop-oriented workflow. It brings execution tools, a built-in script hub, and update management into a single lightweight app for users who want a direct scripting environment on PC.

Rather than emphasizing heavy visuals, the project is centered on the essentials of execution. With multi-language UI support, batch execution, and a persistent session queue, Bunni is structured to make script handling easier for Roblox users working from a local desktop setup.
